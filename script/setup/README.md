# Setup

## System-level dependencies

Setting up system-level requires system-specific scripts with superuser privileges (or under `sudo`).

### Ubuntu 18.04

```bash
$ sudo ./setup.ubuntu-18.04.sh
```

### Amazon Linux 2

```bash
$ sudo ./setup.amazonlinux-2.sh
```
